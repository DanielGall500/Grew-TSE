Grew-TSE Documentation
======================

Welcome to the documentation for **Grew-TSE**. Grew-TSE is a tool for query-based generation of Minimal Pair datasets from treebanks for Targeted Syntactic Evaluation of LLMs. The query language of choice is GREW (Graph Rewriting for NLP).
